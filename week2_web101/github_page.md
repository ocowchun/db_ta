#github page

##usage

###1. 建立名稱為`your-name.github.io`的repo(`your-name`是你的github帳號)

###2. clone repo到你的電腦上

###3. 建立一個`index.html`在該repo的根目錄

```html
<!DOCTYPE html>
<html>
<body>
<h1>Hello World</h1>
<p>I'm hosted with GitHub Pages.</p>
</body>
</html>
```

###4. commit&sync

###5. 前往 `http://your-name.github.io.`瀏覽你的新網站!
