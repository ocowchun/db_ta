##for windowns
install git
install node

####使用 Gitshell
建立資料夾,基本設定
```sh
$ mkdir hello-web
$ npm init
```

##prepare development package
```sh
$ sudo npm install -g generator-simple-page
$ sudo npm install -g bower
```

##init your project
```sh
$ mkdir hello-web
$ yo simple-page
$ gulp
```

##add bower,bootstrap,jquery
```sh
$ bower init
$ bower install bootstrap --save
$ bower install jquery --save
```
